En Promenade

Beaucoup d'animaux marchent. D'autres courent.

Mais certains bébés animaux sont trop petits pour marcher ou courir.

Alors leurs mamans les transportent!

Leurs mamans les transportent sur leur ventre.

Les kangourous sautent. Les bébés kangourous se font porter.

Leurs mamans les transportent dans leur poche.

Les singes marchent. Les bébés singes se font porter.

Leurs mamans les transportent sur leur ventre.

Les koalas grimpent. Les bébés koalas se font porter.

Leurs mamans les transportent sur leur dos.

Les éléphants marchent. Mais les bébés éléphants ne se font pas porter.

Les bébés éléphants marchent à côté de leur maman.
